# 查看响应内容步骤 - 排查识别问题

## 问题

请求成功（200），但显示"未识别到药品信息"，需要查看实际返回的内容。

## 查看响应内容的步骤

### 1. 打开浏览器开发者工具

1. 按 `F12` 或右键 → 检查
2. 切换到 **Network**（网络）标签

### 2. 触发请求

1. 在前端应用中上传图片
2. 点击"开始识别"

### 3. 查看响应内容

1. 在 Network 标签中找到 `recognize` 请求
2. 点击该请求
3. 在右侧面板中，切换到 **Response**（响应）或 **预览**（Preview）标签
4. 查看返回的JSON内容

### 4. 需要关注的信息

查看响应中的以下字段：

```json
{
  "success": true,
  "data": {
    "mergedData": {
      "name": "...",
      "production_date": "...",
      ...
    },
    "recognitionResults": [
      {
        "rawTextPreview": "...",  // 这是豆包API返回的原始文本
        "extractedData": {...}    // 这是解析后的数据
      }
    ],
    "debug": {...}  // 如果有这个字段，说明数据为空
  }
}
```

### 5. 关键信息

请告诉我：
1. **rawTextPreview** 的内容（豆包API实际返回了什么）
2. **extractedData** 的内容（解析后的数据是什么）
3. **mergedData** 的内容（合并后的最终数据）

这样我可以：
- 查看豆包API返回的格式
- 检查解析逻辑是否正确
- 修复解析问题

## 临时调试方案

如果响应中没有调试信息，可以在浏览器控制台运行：

```javascript
// 查看最后一次请求的响应
fetch('https://orange-butterfly-4818.be724115.er.aliyun-esa.net/recognize', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    images: [{
      base64: 'test',
      name: 'test.jpg',
      type: 'image/jpeg'
    }]
  })
})
.then(r => r.json())
.then(data => {
  console.log('完整响应:', JSON.stringify(data, null, 2));
  if (data.data && data.data.recognitionResults) {
    console.log('原始文本:', data.data.recognitionResults[0]?.rawTextPreview);
    console.log('解析数据:', data.data.recognitionResults[0]?.extractedData);
    console.log('合并数据:', data.data.mergedData);
  }
})
.catch(console.error);
```
