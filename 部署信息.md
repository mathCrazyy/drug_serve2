## 🚀 部署步骤

### 1. 边缘函数部署

1. 登录阿里云ESA控制台
2. 创建边缘函数：
   - 函数名：`drug-recognize`
   - 运行时：Node.js 18
   - 入口文件：`index.js`
3. 上传 `edge-function` 目录所有文件
4. 配置环境变量（见下方）
5. 部署并获取函数URL

**环境变量配置**：
```
DOUBAO_API_BASE_URL=https://api.chatfire.site/v1/chat/completions
DOUBAO_API_KEY=your-api-key-here
DOUBAO_MODEL=doubao-1.5-vision-pro-250328
DOUBAO_MAX_TOKENS=1000
DOUBAO_TEMPERATURE=0.1
STORAGE_PREFIX=drug_record:
STORAGE_INDEX_PREFIX=drug_index:
```

### 2. 前端应用部署

1. 构建前端：
   ```bash
   cd frontend
   npm install
   npm run build
   ```

2. 配置API地址：
   - 创建 `frontend/.env` 文件
   - 设置 `VITE_API_BASE_URL` 为边缘函数URL
   - 重新构建

3. 部署到阿里云ESA Pages：
   - 上传 `frontend/dist` 目录所有文件
   - 获取访问URL

### 3. 更新README

部署完成后，更新README.md中的：
- 在线演示URL
- 阿里云ESA Logo图片URL（如需要）
